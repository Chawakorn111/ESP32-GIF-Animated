## Needed
  1. oled 0.96
  2. esp32c3/esp32
  3. touch input

## Pin

             | OLED Pin / Touch Pin | ESP32 Pin |  
    | --------------- | -------------------- | --------- |
    | OLED            | VCC                  | 5v        |
    | OLED            | GND                  | GND       |
    | OLED            | SDA                  | D20       |
    | OLED            | SCL                  | D21       |
    | Touch Pin vcc   | vcc                  | 5v        |
    | Touch Pin       | I/O                  | GPIO3     |
    | Touch Pin GND   | GND                  | GND       |
